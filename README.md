# Tic-Tac-Toe Game in C

This is a simple two-player **Tic-Tac-Toe** game developed in **C language** 
Players take turns to place their marks (X and O) on a 3x3 grid.  
The player who succeeds in placing three marks in a row wins!

---

Game Rules

- The game is played between two players: X and O
- Players select a row and column between (1–3)
- Three marks aligned horizontally, vertically, or diagonally wins the game
- If all cells are filled and no winner → Draw
